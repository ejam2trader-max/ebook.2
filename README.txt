GracePrint PWA LITE | Rut y Boaz

Este paquete está optimizado para subir a GitHub (web) sin archivos pesados.
Las imágenes fueron convertidas a .webp y redimensionadas para reducir peso.

Uso (GitHub Pages)
1) Sube el contenido de esta carpeta al repo (público).
2) Ve a Settings > Pages > Deploy from branch (main / root).
3) Abre la URL de GitHub Pages.

Instalar en móvil
- Abre la URL en Chrome (Android) o Safari (iOS) y elige "Add to Home Screen".

PDF
- En el libro pulsa el botón "PDF" (imprime / guarda como PDF).
